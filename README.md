# ucb-eecs-servers

Current usage data for UC Berkeley's EECS servers.

## Usage

### Set Up Remote Update

Edit the information in `bin/config.sh` and then execute:

    bash bin/setup.sh

### Remote Update

    bash bin/update.sh

### Run Server

    node server.js

This will also run remote update every 5 minutes.
